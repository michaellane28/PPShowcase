PowerplayShowcase is a project created to display certain games from the NHL 2024-25 season. This web application uses the official schedule dataset from NHL.com and web scrapes player and team information from DailyFaceoff.com. Each day, certain games are picked from the schedule dateset and added to the websites list for that day. That day is also then included in the HTML code and displayed using the date slider.

The criteria that a game needs to meet is as follows: One of the teams must have been in the top 5 in pentalties per game in the previous season:

And the other team must have at least two players who are on both the 1st forward line and the 1st powerplay line.
